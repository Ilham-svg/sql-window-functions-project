# SQL Window Functions Project

This project demonstrates advanced analytical SQL using PostgreSQL window functions.
All analysis is performed without GROUP BY in final outputs.

## Key Topics
- Latest record per group
- Ranking & top-N analysis
- Running totals & moving averages
- Trend analysis using LAG
- Deduplication using ROW_NUMBER

## Tools
- PostgreSQL
- pgAdmin 4
- Git & GitHub

## PROJECT RULES

- No GROUP BY in final queries
- Use ROW_NUMBER, RANK, DENSE_RANK, SUM OVER, COUNT OVER, LAG, AVG OVER
- Use CTEs for clarity
- PostgreSQL syntax
